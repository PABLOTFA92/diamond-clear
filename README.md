# Diamond Clear (Starter)

Este projeto é um **starter** do app por link (PWA) da Diamond Clear, com:
- Telas cliente/admin (demo)
- Manifest PWA + ícones
- Botão WhatsApp com o número 43 99838-7306

## Rodar no PC
1) Instale Node.js (LTS)
2) Abra o terminal na pasta do projeto e rode:
   npm install
   npm run dev

Abra: http://localhost:3000

## Publicar na Vercel (grátis)
1) Suba este projeto para o GitHub
2) Na Vercel: Add New → Project → selecione o repo → Deploy
